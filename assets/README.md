# Thirdeye Visual Assets

This folder contains branding assets for Project Thirdeye.

## Background Images

### blue-cosmic-eye-bg.png (Subtle Version)
**Use for**: Chat UI backgrounds, web interface, dark mode applications

**Characteristics**:
- Pure black background (#000000)
- Soft cyan-blue glowing eye (#00d4ff)
- Very low opacity (20%) for text readability
- Calm, watchful, professional aesthetic

**CSS Implementation**:
```css
.chat-container {
  background: #000 url('/assets/blue-cosmic-eye-bg.png') center/cover fixed;
  background-blend-mode: overlay;
}
```

### blue-cosmic-eye-dramatic.png (Bold Version)
**Use for**: Loom thumbnails, social media, project covers, marketing materials

**Characteristics**:
- Pure black background (#000000)
- Bright cyan-blue eye with strong radial glow
- Layered rings like a lens or chakra
- Higher visibility (30% opacity) for impact
- Eye-catching, powerful, cosmic aesthetic

**Usage Tips**:
- Loom: Use as virtual background or video thumbnail
- Build Club: Upload as project cover image
- GitHub: Use in README header
- Twitter: Use for announcement posts

## Color Palette

- **Primary Blue**: #00d4ff (cyan-blue glow)
- **Background**: #000000 (pure black)
- **Accent**: #0066ff (deep cosmic blue)
- **Nebula**: Dark blues/purples (#001122, #1a0033)

## Symbolism

The **third eye** represents:
- **Insight**: Seeing what others miss (blind spots, failure modes)
- **Foresight**: Pre-mortem analysis before shipping
- **Truth**: Adversarial honesty vs polite AI
- **Wisdom**: Grounded in 300+ expert podcast transcripts
